# CardVector-site

Public website for Putnam Collectibles at https://cardvector.app.

This is a static GitHub Pages site. The home page is customer-facing for Putnam Collectibles; CardVector is presented as the technology powering the business.

## GitHub Pages

Recommended Pages settings:

- Source: Deploy from a branch
- Branch: main
- Folder: /(root)
- Custom domain: cardvector.app

## Routes

The site includes a GitHub Pages `404.html` fallback so CardVector QR URLs can resolve client-side:

- `/etb/ETB-001`
- `/location/ETB-001/A`
- `/lot/LOT-2026-001`

Future public pages are reserved but not fully built yet:

- `/buylist`
- `/bulk`
- `/events`
- `/contact`
- `/about`
